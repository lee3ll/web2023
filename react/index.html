REACT

React는 Facebook에서 개발한 자바스크립트 라이브러리로, 사용자 인터페이스를 만들기 위해 사용됩니다.

React는 화면을 여러 개의 컴포넌트로 분할하고, 이러한 컴포넌트들을 조합하여 복잡한 UI를 만들 수 있도록 도와줍니다. React는 Virtual DOM을 사용하여 화면을 관리하며, 이를 통해 빠르고 효율적인 UI 업데이트를 가능하게 합니다. 또한 React는 JSX라는 문법을 제공하여 HTML과 유사한 구문으로 UI를 작성할 수 있게 해줍니다.
