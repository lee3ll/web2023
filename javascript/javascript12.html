01. 1부터 100까지 출력하기

for문을 1부터 100까지 출력하는 예제입니다.

{
    for(let i=1; i<=100; i++){ 
    
    console.log(i)
    }
    
}
결과 확인하기
1
2
3
~~
100

03. 1부터 100까지 짝수만 출력하기

for문을 짝수만 출력하는 형식입니다. (i+=2)는 i = i + 2 의 약식입니다.

{
    for(let i=0; i<=100; i+=2){
    
    console.log(i)
    }
