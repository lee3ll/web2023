배열 메서드 : push()/ pop()

push() 메서드는 배열 마지막 요소에 요소를 추가하고, pop() 메서드는 배열 마지막 요소를 삭제합니다.


    const arrNum = [100, 200, 300, 400, 500];
    const arrPush = arrNum.push(600);

    console.log(arrNum)   //[100, 200, 300, 400, 500, 600]


    const arrNum1 = [100, 200, 300, 400, 500];
    const arrPush1 = arrNum1.pop();

    console.log(arrNum1)   //[100, 200, 300, 400]

배열 메서드 : unshift()/ shift()

unshift() 메서드는 배열 처음 요소에 요소를 추가하고, shift() 메서드는 배열 처음 요소를 삭제합니다.


    const arrNum = [100, 200, 300, 400, 500];
    const arrPush = arrNum.unshift(600);

    console.log(arrNum)   //[600, 100, 200, 300, 400, 500]


    const arrNum1 = [100, 200, 300, 400, 500];
    const arrPush1 = arrNum1.shift();

    console.log(arrNum1)   //[200, 300, 400, 500]

배열 메서드 : join()

join() 메서드는 배열 요소를 문자열로 결합합니다. 반환은 문자열입니다.


const arrNum = [100, 200, 300, 400, 500];
const text1 = arrNum.join('');  //100200300400500
const text2 = arrNum.join(' ');  //100 200 300 400 500
const text3 = arrNum.join('*');  //100*200*300*400*500
const text4 = arrNum.join('-');  //100-200-300-400-500
const text5 = arrNum.join('🍧');  //100-200-300-400-500

console.log(text4)
